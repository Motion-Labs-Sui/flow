## üîß Setup

### Prerequisites
- Node.js 18+ or Bun
- Claude API key from Anthropic
- Sui private key for Walrus deployment

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/motion-labs/flow-vce.git
   cd flow-vce
   ```

2. **Install dependencies**
   ```bash
   bun install
   # or
   npm install
   ```

3. **Start development server**
   ```bash
   bun run dev
   # or
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Configuration

1. Click the **Settings** button in the top-right corner
2. Enter your **Claude API Key** (`sk-ant-api03-...`)
3. Enter your **Sui Private Key** for Walrus deployment
4. Select your preferred **Network** (testnet/mainnet)
5. Save settings

## üéÆ How to Use

### 1. **Create a Project**
- Click "New Project" or start from the Projects tab
- Give your project a meaningful name

### 2. **Generate a Website**
- Navigate to the "Generate" tab
- Write a detailed prompt describing your desired website
- Click the magic wand button or press ‚åò+Enter
- Watch Claude create your website in real-time!

### 3. **Preview & Edit**
- Switch to the "Preview" tab to see your website
- Toggle between desktop, tablet, and mobile views
- Switch to "Code" view to inspect the generated code
- Copy individual files or download the complete HTML

### 4. **Deploy to Walrus**
- Click "Deploy to Walrus" in the preview panel
- Your site will be uploaded to Walrus Sites
- Get a unique `.wal.app` URL for your decentralized site
- Share your site with the world!
